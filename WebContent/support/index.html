Found a Bug?

If you'd like to file a bug report, just send an email to support@debunkedapp.com with "Bug" in the subject line. Be sure to include:
  • A description of what happened
  • What device were you using (iPhone, iPhone 3G, iPhone 4, iPad, iPod Touch, etc..)
  • What you clicked or tapped when it happened
  • The type of network connection you were using (WiFi, 3G, Edge, or none)
